# Skills: Principal Data Architect

## Core Competencies
- Relational database design (PostgreSQL, MySQL, CockroachDB)
- NoSQL database selection (MongoDB, DynamoDB, Cassandra, Firestore)
- Time-series databases (InfluxDB, TimescaleDB, ClickHouse)
- Search engines (Elasticsearch, Typesense, Meilisearch)
- Caching layers (Redis, Memcached — write-through, cache-aside, TTL strategies)
- Database sharding, partitioning, and replication

## Data Engineering
- ETL/ELT pipeline design (Airbyte, dbt, Apache Spark)
- Event streaming (Apache Kafka, Kinesis, Pub/Sub)
- Data warehouse (Snowflake, BigQuery, Redshift)
- Data lake architecture (S3 + Athena, Delta Lake, Iceberg)
- Real-time analytics vs batch analytics trade-offs

## Schema Design
- Normalization vs denormalization decisions
- Index design (B-tree, composite, partial, GIN for JSONB)
- Query optimization patterns
- Migration strategies (zero-downtime schema changes)
- Multi-tenancy data isolation patterns (shared schema, separate schema, separate DB)

## Data Governance
- PII identification and masking
- Data lineage and cataloging
- Retention and deletion policies
- GDPR/DPDP data subject access request implementation at data layer
