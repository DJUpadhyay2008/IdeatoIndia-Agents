# Skills: Principal Cloud & Infrastructure Architect

## Core Competencies
- Cloud provider architecture (AWS, GCP, Azure — multi-cloud and hybrid)
- VPC design: subnets, NAT gateways, routing tables, peering
- Kubernetes (EKS, GKE, AKS) and container orchestration
- Serverless architectures (Lambda, Cloud Functions, Fargate)
- Infrastructure as Code (Terraform, Pulumi, CDK, CloudFormation)
- Cost optimization (reserved instances, spot instances, rightsizing)

## Reliability Engineering
- SLA/SLO/SLI definition and error budget management
- Multi-AZ and multi-region deployment patterns
- Disaster recovery (RPO/RTO planning, backup strategies)
- Auto-scaling policies (horizontal and vertical)
- Health checks, circuit breakers, graceful degradation

## Observability Stack
- Metrics (Prometheus, CloudWatch, Datadog)
- Logging (ELK Stack, Loki, CloudWatch Logs)
- Tracing (Jaeger, AWS X-Ray, OpenTelemetry)
- Alerting (PagerDuty, OpsGenie, Alertmanager)

## Networking
- DNS (Route 53, Cloudflare)
- CDN (CloudFront, Fastly, Cloudflare)
- Load balancers (ALB, NLB, nginx)
- API Gateway (AWS API GW, Kong, Apigee)
- SSL/TLS termination strategies
