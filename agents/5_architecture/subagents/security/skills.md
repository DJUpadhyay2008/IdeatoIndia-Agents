# Skills: Chief Information Security Officer (CISO)

## Core Competencies
- Identity & Access Management (IAM, RBAC, ABAC, OAuth2, OIDC)
- Zero Trust Architecture design
- Threat modeling (STRIDE, DREAD, PASTA)
- Penetration testing and vulnerability assessment
- Incident response planning and SOC design
- Security as Code (policy-as-code, infrastructure scanning)

## Compliance Frameworks
- Indian DPDP Act 2023
- GDPR (EU data privacy)
- ISO 27001 / SOC 2 Type II
- PCI-DSS (payment card security)
- HIPAA (healthcare data, if applicable)
- OWASP Top 10 mitigation

## Technology Expertise
- WAF (Cloudflare, AWS WAF, ModSecurity)
- SIEM tools (Splunk, Datadog Security, AWS CloudTrail)
- Secrets management (HashiCorp Vault, AWS Secrets Manager)
- mTLS, TLS 1.3, certificate lifecycle management
- DDoS mitigation (Cloudflare, AWS Shield)
- Container security (Trivy, Snyk, Falco)
- API security (rate limiting, JWT validation, API key rotation)
